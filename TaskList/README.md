# 📋 Список задач — React приложение для управления задачами

## 📌 Описание проекта

**Список задач** — это одностраничное клиентское веб-приложение, разработанное на React, которое позволяет управлять списком задач:

- просматривать список задач
- искать задачи по названию
- фильтровать по статусу
- добавлять новые задачи
- редактировать существующие
- просматривать подробную информацию

Проект создан в учебных целях для освоения базовых практик современной frontend-разработки.

---

## 🎯 Цель проекта

Освоить ключевые навыки разработки клиентских приложений:

- создание проекта на React
- работа с компонентами
- управление состоянием (state)
- передача данных между компонентами
- использование пользовательских хуков
- работа с формами и валидацией
- фильтрация и отображение данных
- обработка загрузки и ошибок
- базовая архитектура проекта

## ⚙️ Функциональность

### 📋 Работа с задачами

- Отображение списка задач
- Просмотр подробной информации
- Добавление новой задачи
- Редактирование задачи

### 🔍 Поиск и фильтрация

- Поиск по названию
- Фильтрация по статусу:
    - Все
    - Не начато
    - В работе
    - Выполнено

### ⏳ Состояния приложения

- Индикатор загрузки (имитация API)
- Обработка ошибок загрузки
- Пустой список / нет результатов

### 🧾 Формы

- Валидация полей:
    - обязательное название
    - минимальная длина
- Отображение ошибок
- Поддержка режима редактирования

---

## 🔄 Работа с данными

В проекте используется имитация API:

- данные хранятся в `data/tasks.js`
- загрузка реализована через `setTimeout`
- реализованы состояния:
    - `loading`
    - `error`
